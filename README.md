## üìù Exam Syllabus: SY0-701

The preparation materials in this repository align with the official CompTIA Security+ (SY0-701) exam objectives, covering the following five domains:

1.  **General Security Concepts (12%)**
    * Security controls, fundamental security concepts (CIA triad), and change management processes.
2.  **Threats, Vulnerabilities, and Mitigations (22%)**
    * Threat actors, vectors, vulnerability management, and mitigation strategies.
3.  **Security Architecture (18%)**
    * Security implications of different architecture models, hardware/software infrastructure, and data protection.
4.  **Security Operations (28%)**
    * Monitoring, incident response, disaster recovery, and account management.
5.  **Security Program Management and Oversight (20%)**
    * Governance, risk management, compliance, and third-party risk assessment.
